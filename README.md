# Kosmos Travel

This project is a full-stack travel booking application with a Spring Boot backend and Angular frontend.  
It includes user registration, tour packages, bookings, payments, and admin management.

## Features
- User registration and login
- Tour package browsing and booking
- Payment integration
- Admin dashboard for managing bookings and users

## Technologies Used
- Backend: Java, Spring Boot, Hibernate, MySQL
- Frontend: Angular, TypeScript, HTML, CSS
- Build Tools: Maven (backend), npm (frontend)

## How to Run the Project

### Backend
1. Make sure MySQL server is running and the database `kosmosdb` is created.
2. Update your `application.properties` with your database username and password or environment variables.
3. Open the `backend` folder in Spring Tool Suite or your preferred IDE.
4. Run the application as a Spring Boot App (`TravelKosmosApplication.java`).
5. The backend server will run on `http://localhost:8080`.

### Frontend
1. Open the `frontend` folder in VS Code or any code editor.
2. Run `npm install` to install all dependencies.
3. Run `ng serve -o` to start the Angular development server.
4. The frontend will open automatically in your default browser at `http://localhost:4200`.
